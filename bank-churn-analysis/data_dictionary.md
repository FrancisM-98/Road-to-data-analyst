# Churn Analysis - Data Dictionary

This document provides a description of each column in the Churn Modelling dataset.

| Column Name | Description |
| :--- | :--- |
| **RowNumber** | The specific row number (index) in the dataset. |
| **CustomerId** | Unique identifier for identifying the customer. |
| **Surname** | The last name of the customer. |
| **CreditScore** | The customer's credit score. |
| **Geography** | The country where the customer resides (e.g., France, Spain, Germany). |
| **Gender** | The customer's gender (Male, Female). |
| **Age** | The age of the customer. |
| **Tenure** | The number of years the customer has been with the bank. |
| **Balance** | The customer's account balance. |
| **NumOfProducts** | The number of bank products the customer uses (e.g., savings account, credit card). |
| **HasCrCard** | Indicates whether the customer has a credit card (1 = Yes, 0 = No). |
| **IsActiveMember** | Indicates whether the customer is an active member (1 = Yes, 0 = No). |
| **EstimatedSalary** | The customer's estimated annual salary. |
| **Exited** | **Target Variable.** Indicates whether the customer has churned (left the bank) (1 = Yes, 0 = No). |
